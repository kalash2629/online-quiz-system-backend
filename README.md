ONLINE QUIZ SYSTEM BACKEND -->

A robust and scalable backend system for an Online Quiz Platform built using Spring Boot. This project provides RESTful APIs for managing quizzes, users, authentication, and results with secure role-based access.

🚀 Features -->
🔐 JWT-based Authentication & Authorization
👨‍🏫 Admin functionalities (create/manage quizzes & questions)
👨‍🎓 Student functionalities (attempt quizzes & view results)
📊 Result evaluation and tracking
🌐 RESTful API architecture
🛡️ Spring Security integration
⚡ Docker support for easy deployment

🏗️ Tech Stack -->
Backend: Spring Boot
Language: Java
Database: MySQL (configurable)
Security: Spring Security + JWT
Build Tool: Maven
Containerization: Docker & Docker Compose
